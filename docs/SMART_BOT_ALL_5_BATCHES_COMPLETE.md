## **AI Decision Flow**

```
Every Frame:

1. DETECT
   ├─ Scan for hazards (300px ahead)
   ├─ Scan for enemies (400px ahead)
   ├─ Scan for pickups (250px around)
   └─ Check advanced hazards (lightning, water, etc.)

2. ANALYZE
   ├─ Calculate threat levels
   ├─ Assess resource status
   ├─ Plan pathfinding
   └─ Predict platform safety

3. DECIDE (5-Tier Priority)
   ├─ Priority 1: Health < 30%? → SEEK_HEALTH
   ├─ Priority 2: Hazard < 150px? → AVOID
   ├─ Priority 3: Enemy < 250px? → ENGAGE
   ├─ Priority 4: Pickups visible? → COLLECT
   └─ Priority 5: Default → SPRINT_NORMAL

4. ACT
   ├─ Set output flags (ShouldJump, ShouldAttack, etc.)
   ├─ Calculate optimal jump height
   ├─ Plan next move
   └─ Inject real keys

5. EXECUTE
   └─ Game engine processes keys normally
```
