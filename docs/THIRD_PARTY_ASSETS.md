# Third-Party Asset Imports (Legally Clean)

All imported assets in this pipeline are original third-party assets from approved open sources.
No Nintendo-owned names, sprites, ripped art, ripped sounds, or ripped level layouts are included.

## Import Policy
- Runtime imports are copied under `Assets/third_party/vendor/...` (custom engine fallback root).
- Original archives are kept in-repo under `third_party/archives/...`.
- Raw extracted files are kept under `third_party/extracted/...`.
- Custom project art is never overwritten by the import scripts.

## Pixel-Art Filtering / Pixel-Perfect Guidance

This repository is a custom WinForms game (non-Unity/Godot/web runtime).
Use nearest-neighbor rendering for imported pixel textures in runtime draw code where images are scaled.
No risky bulk import-metadata edits are applied by these scripts.
